# sparam 总设计文档（索引）

本文件作为项目设计文档的总入口，具体模块设计已拆分为独立文档。

## 1. 项目概述

sparam 是一款 MCU 参数监测和标定工具，通过统一协议连接上位机与下位机，支持 GUI 与 CLI 两种工作模式。

核心目标：

- 零代码侵入：直接解析 ELF/MAP 获取变量地址
- 传输无关：协议与串口/Socket 通道解耦
- 多设备支持：设备 ID 范围 0-255
- 双界面：GUI 与 CLI 兼容脚本化与交互式场景

## 4. 当前阶段重点

### 4.1 协议

协议帧格式、命令编码、数据类型、应答机制与通信流程，请见 `协议设计文档.md`。

### 4.2 GUI

当前 GUI 已支持连接管理、变量浏览、监测、Read Once / Write Once。

Issue #3（M3）已确认目标：

- 默认无圆角
- 左 Sidebar / 中 Waveform / 右 Inspector
- 子面板 Dock 化（可停靠、可拉伸）
- 布局持久化（重启恢复）为必做项

详细说明请见 `sparam-gui设计文档.md`。

### 4.3 CLI

CLI 负责自动化与脚本化操作，提供端口、读写、监测、停止等命令。

详细说明请见 `sparam-cli设计文档.md`。
